## 下载配置文件`dotfiles`

```bash
git clone https://github.com/ChongYanOvO/dotfiles ~/dotfiles
```

## 使用`stow`来管理配置文件

`stow`GitHub链接🔗: [stow](https://github.com/aspiers/stow)

- 下载(使用`brew`)

```bash
brew install stow
```

- 示例: 配置 zsh

```bash
cd ~/dotfiles
stow zsh
```

## `wezterm`配置

`wezterm`官网🔗: [wezterm](https://wezfurlong.org/wezterm/)

- 下载(使用`brew`)

```bash
brew install --cask wezterm
```

- 配置使用

```bash
cd ~/dotfiles
stow wezterm
```
